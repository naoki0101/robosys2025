# plusコマンド

![test](https://github.com/naoki0101/robosys2023/actions/workflows/test.yml/badge.svg)

 標準入出力から読み込んだ数字を足す。


## 実行方法
＊ seqなどを使用して数字を標準入出力して./plusで実行可能です。

## 実行例
＊ seq 10 | ./plus

　55

　　 のようなことができます。

## 必要なソフトウェア
　　　＊ python

                 テスト済み： 3.7～3.10

## テスト環境
＊　Ubuntu 24.04
